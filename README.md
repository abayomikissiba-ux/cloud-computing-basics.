# Cloud Computing Basics

## Cloud Provider
Amazon Web Services (AWS)

## Service Used
Amazon EC2 (Elastic Compute Cloud)

## Objective
To compare the pricing of a cloud computing service across different geographic regions using the AWS Pricing Calculator.

## Regions Compared

| Region | Monthly Cost | Yearly Cost |
|------|------|------|
| US East | $1.53 | $18.36 |
| Africa (Cape Town) | $3.94 | $47.28 |
| Europe (London) | $2.26 | $27.12 |
| US West | $2.12 | $25.44 |
| South America | $4.30 | $51.60 |
| Asia Pacific (Thailand) | $7.08 | $84.96 |

## Observations
The cost of running the same EC2 instance varies across regions. This variation occurs due to factors such as infrastructure cost, electricity, demand, and operational expenses in each location.

## Lessons Learned
- Cloud computing services are priced differently depending on region.
- Pricing calculators help estimate infrastructure cost before deployment.
- AWS provides flexible infrastructure across multiple geographic regions.
